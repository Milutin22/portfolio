Cigar.com

details:

Objective

Design and code two web pages, while following the visual rules when composing elements. It was my first project and it was allowed to use only HTML, CSS, and JavaScript without any frameworks. I specifically dealt with the topic related to Cuban cigars. Since this project was my first, it is not responsive.

Goals

-Using a network or grid whose number of columns is divisible by 3;
-The color palette in accordance with the theme;
-The functionality of all parts of the site;
-Design in line with trends;
-Subtle animations that enhance interaction;
-Linked parts of the site with real online stores;

Technologies

-Adobe Photoshop;
-Adobe Illustrator;
-HTML 5;
-CSS 3;
-Java Script;
-Mozilla Firefox code inspector;
