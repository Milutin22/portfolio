Paintball Belgrade

details:

Objective

Do the design and complete front end part of the site. I used BEM(Blocks, Elements and Modifiers) with which I had better workflow, and of course SASS was my best tool to organize code. On this website you can find out more about this imaginary company called the Paintball Belgrade.

Goals

-Responsive for desktop, tablet, and mobile screens;
-Design in line with trends;
-Subtle animations that enhance interaction;
-Making a navigation bar that is designed for desktop and mobile phone dimensions;
-Inserting a realistic map with the help of Leaflet;
-The color palette in accordance with the theme;
-The functionality of all parts of the site;

Technologies

-Adobe Suite;
-HTML 5;
-Bootstrap 4 with CSS flexbox;
-CSS 3(SCSS);
-Java Script with jQuery;
-Mozilla Firefox and Google code inspector;
