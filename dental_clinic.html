Dental Clinic

Objective

Do the design and complete front end part of the site. I did this project on my own initiative where I first encountered bootstrap. As part of this project, I established knowledge of responsive design. You can make appointments on the site, leave comments, as well as find out more information about the provision of services.

Goals

-Design in line with trends;
-Subtle animations that enhance interaction;
-Responsive for desktop, tablet, and mobile screens;
-Inserting a realistic map with the help of Leaflet;
-The color palette in accordance with the theme;
-The functionality of all parts of the site;
-Making a form;

Technologies

-Adobe suite;
-HTML 5;
-Bootstrap 4 with CSS flexbox;
-CSS 3;
-Java Script;
-Mozilla Firefox code inspector;
